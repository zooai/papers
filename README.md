## Building Papers

All papers are automatically compiled to PDF on every commit via GitHub Actions.

```bash
# Install LaTeX (one-time)
brew install basictex  # macOS

# Compile all papers
make all

# View PDFs (macOS)
make view

# Clean intermediate files
make clean
```

Download PDFs: https://zooai.github.io/papers/
